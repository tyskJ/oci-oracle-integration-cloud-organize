統合作成
=====================================================================
* コンソールアクセス

.. image:: ./doc/image/pic01.png
  :scale: 100%

* ``ハンバーガーマーク`` → ``設計`` をクリック

.. image:: ./doc/image/pic02.png
  :scale: 100%

* ``統合`` → ``作成`` をクリック

.. image:: ./doc/image/pic03.png
  :scale: 100%

* ``スケジュール`` をクリック

.. image:: ./doc/image/pic04.png
  :scale: 100%

* 適当な名前を入力し ``作成`` をクリック

.. image:: ./doc/image/pic05.png
  :scale: 100%

* 矢印上の ``+`` マークをクリックし、 ``OCI関数`` をクリック

.. image:: ./doc/image/pic06.png
  :scale: 100%

* 該当のFunctionsを選択し、 ``続行`` をクリック

.. image:: ./doc/image/pic07.png
  :scale: 100%

* 何かしら入力しないといけないので、適当に入力し ``続行`` をクリック

.. image:: ./doc/image/pic08.png
  :scale: 100%

* 同様に何かしら入力しないといけないので、適当に入力し ``続行`` をクリック

.. image:: ./doc/image/pic09.png
  :scale: 100%

* 最後確認画面がでるので ``終了`` をクリック

.. image:: ./doc/image/pic10.png
  :scale: 100%

.. note:: 

  * 同様の手順で ``compute-stop`` Functions を後段に登録します。
